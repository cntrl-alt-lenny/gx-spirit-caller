# dsd named-function signatures

`nitro-sdk-eur/` contains the 67 YAML signatures generated by `dsd sig new`
from the uniquely named EUR ARM9 functions in the checked-in config. The
signatures are used with `dsd sig apply --dry` to identify un-named regional
twins; the resulting symbol changes are applied through
`tools/rename_symbol.py --cascade`.

Two named functions were omitted because `dsd sig new` reported multiple
function instances and requires an explicit index: `Task_PostLocked` and
`Task_InvokeLocked`.
