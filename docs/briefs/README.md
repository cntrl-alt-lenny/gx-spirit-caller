## Adding a new brief

1. Create `docs/briefs/NNN-slug.md` with the next number.
2. First line: `### <branch>/<slug>` (matches the working branch name).
3. First paragraph after the heading: `**Goal:** ...` — one or two
   sentences describing what success looks like.
4. `python tools/generate_briefs_index.py` to refresh this file.
5. Commit both the new brief and the updated `README.md`.
