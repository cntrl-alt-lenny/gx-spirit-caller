## Closest matched example

**Path:** src/overlay002/func_ov002_022b3910.c
**Why similar:** Same early-exit null-guard pattern on a self-struct field followed by
a bounded loop over slot indices calling ov002 helper pairs.
**Key lesson:** The two early-exit guards (`addeq sp; ldmeqia`) must be written as explicit
`if (cond) return;` statements — mwcc fuses the `add sp` + `ldmia pc` into a single epilogue
only when there is no work between the guard and the return; using a goto or a nested block
will split the epilogue and misalign the instructions. The fixed-point rounding sequence
`(val + (val >> 27)) >> 5` must be kept as written; folding the rounding into a simple division
will emit a different shift sequence.
