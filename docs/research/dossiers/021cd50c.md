## Closest matched example
**Path:** `src/overlay011/func_ov011_021cc9b4.c`
**Why similar:** 2-case switch on `data_ov011_021d4000+0x2a0`; each arm copies a
disjoint 3-word struct block (no overlap between cases).
**Key lesson from that file:** The 2-case switch with no default falls through to the
`break` in each arm — mwcc emits a `cmp/bne` chain rather than a jump table when
there are only two arms. Write `switch(…){ case 0:…break; case 1:…break; }` with no
explicit `default:` and let the compiler handle the fallthrough as a silent no-op;
adding an explicit `default: break;` may change the epilogue code shape.
