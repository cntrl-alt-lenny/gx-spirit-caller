## Struct context

The card pointer `sl` (r0 arg) holds `f0` at [sl] and `f2` at [sl,#0x2]. Bit 0 of `f2` (`lsl #0x1f / lsr #0x1f`) gives a player index (0 or 1); `rsb r1, r0, #1` inverts it to select the opposing player's row. The row pointer is `data_ov002_022cf16c + (player & 1) * 0x868`. Each slot is `+0x14` stride; the slot walk runs `s=0..4` (5 slots), advancing `r9` by `+0x14` each iteration and using `r8` as the slot counter.
