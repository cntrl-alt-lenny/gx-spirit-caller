The `tst+movne` pattern: MWCC emits `tst rN, mask; movne rOut, #val` (or `orrne`)
for `if (x & mask) result = val`. Do NOT write as `result = (x & mask) ? val : other`
which can emit a different conditional sequence.

changes the branch structure.
