**New verdict:** tractable — the shift chain is the documented mwcc2.0/sp1p5 idiom for
signed division-by-32 + remainder. `func_0202b0b4` returns a card sort key in r0 (used
as `key * 8` offset into `data_ov006_0224f3e0+4`). The `asr #0x4 / lsr #0x1b / asr #0x5`

0x1f`. Final `and r0, r0, r2, lsl r1` extracts one bit. The function is a compact
bit-test accessor — no calls after the first `bl func_0202b0b4`.

4. `func_0202b0b4` takes `p` (arg0 of the outer function); arg1 `slot` goes to `r4`.
   The `mov r4, r1` at entry confirms `slot` is arg1, `p` is arg0.
