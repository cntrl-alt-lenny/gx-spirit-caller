Div-magic notes:
- `/32` with round-toward-zero: mwcc emits `asr r,r,#31` (sign extend) then `rsb` + `lsl#27` + `ror#27` pattern for signed `/32`, then `asr#5`. Do not write `>> 5` — write `/ 32`.
- `%8` signed: mwcc emits `lsl#0x1b` + `ror#0x1b` (which extracts bits 2:0 with sign extension for signed mod). Write `% 8` on a signed int.
- `abs`: `movpl` / `rsbmi` — write `if (x < 0) x = -x;` or `ad = (ad < 0) ? -ad : ad;`; do not use stdlib `abs()`.
