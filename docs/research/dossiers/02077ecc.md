## Closest matched example

**Path:** src/main/func_0207d18c.c
**Key lesson:** when ctx is bound to a callee-saved register (r8), mwcc generates a single `ldr ctx_reg, [sp+N]` at entry and reuses it across all field accesses. If any computed address (`&ctx->buf[byte_off]`) is passed as an argument to a call, mwcc must recompute the address from r8 after each call because calls may clobber the offset register — so never pre-bind `&ctx->buf[byte_off]` to a local that crosses a call boundary.
