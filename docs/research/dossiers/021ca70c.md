- `lsl #0x1f / lsr #0x1f` for bit-0 extract at f17c: write as `f17c & 1`.
- Veneer BL `.word 0xebff9b06` — look up the target symbol in `config/eur/overlay013/symbols.txt`; declare it as an extern and call it normally. The linker will insert a veneer for the out-of-range BL.
- Return value: `movne r0, #0x1 / moveq r0, #0x0` → `return anim_grp != NULL`.
