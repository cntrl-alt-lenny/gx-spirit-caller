## Struct context

Takes `(int player, int slot)`. Calls `func_ov002_021b9ecc(player, slot)` to get slot_id. If 0, return 0. If `slot_id == 0x16f8`: reads from `data_ov002_022cf1ac` (flag word array) at `player*0x868 + slot*0x14` offset and `data_ov002_022cf1a4` (u16 array) at same offset; runs occupancy test; if passes return 0. Otherwise: calls `func_0202e2f8(slot_id)`, then `func_ov002_021ca51c()` for a flags bitmask, then bit-tests flags 0x1/0x2/0x4/0x8. The bit 0x4 and 0x8 checks read `Ov002Slot.id_lo13` bit-13 (`lsl#18/lsr#31`) from the requesting player's slot to check orientation.
