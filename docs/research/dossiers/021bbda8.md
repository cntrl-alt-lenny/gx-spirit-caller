## Struct context

An additional bit test `(word << 0x12) >> 0x1f` extracts bit 13 (one above id_lo13) as a filter. Then walks the extra-slot list (count at `data_ov002_022cf180[player*0x868/4]`, entries at `cf16c + player*0x868 + 0x418`) doing a plain `id_lo13` extract and test.
