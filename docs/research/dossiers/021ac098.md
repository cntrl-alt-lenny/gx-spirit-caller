```c
int func_ov000_021ac098(HitRect *r, int x, int y) {
    func_ov000_021abe64(r);   /* unpack/setup (return value discarded) */

    unsigned int w0 = r->word0;
    /* Y range: [lo_y, lo_y + y_size*2) */
    int lo_y   = (int)((w0 << 0x11) >> 0x17);   /* bits 14-21, *2 */
    int y_size = (int)(w0 >> 0x18);              /* bits 31-24 */
    if (x < lo_y || x >= lo_y + y_size * 2)
        return -1;

    /* X range: [x_lo, x_lo + x_size*2) */
    int x_lo   = (int)((w0 << 0x8) >> 0x17);    /* bits 7-14, *2 */
    if (y < x_lo)
        return -1;

    unsigned int w1 = r->word1;
    int x_size = (int)((w1 << 0x18) >> 0x18);   /* bits 7-0 */
    int x_hi   = x_lo + x_size * 2;
    if (y >= x_hi)
        return -1;

    /* Zone 1/2 split */
    int zw1    = (int)((w1 << 0x10) >> 0x18);   /* bits 15-8 */
    int z1_hi  = x_lo + zw1;
    if (y < z1_hi) return 1;
    if (y < x_lo)  return 2;   /* effectively y < z1_hi already handled → next boundary */

    /* Zone 3 */
    int zw_extra = (int)((w1 << 0x8) >> 0x18);  /* bits 31-24 */
    if (y < z1_hi + zw_extra) return 3;          /* note: boundary recalculated */

    /* Zone 4 vs 5 */
    if (y < x_hi - zw1) return 4;
    return 5;
}
```

**Levers:**
- `lsl #0x11 / lsr #0x17` = extract bits 14-21 as unsigned: `(w0 & 0x3fc000u) >> 13` in one view, but mwcc prefers the two-shift form.
- `lsl #0x8 / lsr #0x17` for the x_lo field — same idiom.
- `movcc r0, #0x1 / ldmccia` → `if (y < threshold) return 1;` — keep early-return branches as separate `if` / `return` pairs.
- `movlt r0, #0x2 / ldmltia` → signed-less-than branch; the test is `cmp r4, r0` where `r0` holds the boundary from the previous instruction, not a fresh load.
- `movcc r0, #0x4 / movcs r0, #0x5 / ldmia` → the zone 4/5 selector: a single `if (y < boundary) return 4; return 5;` using the same condition register.
- `.L_a4: mvn r0, #0x0` → return -1 = `~0`; use `return -1` which mwcc compiles to `mvn r0, #0`.
