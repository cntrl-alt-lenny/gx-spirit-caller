**Why similar:** Uses `*(struct blk3*)dst = *(struct blk3*)src` to generate
`ldm/stm` (stmia) for a 3-word copy, in a switch-selected direction.
**Key lesson from that file:** The struct-copy idiom `*(Blk3*)dst = *(Blk3*)src`
is the ONLY reliable way to get `ldm/stm` — do not write three separate `int`
assignments. The struct must be declared with exactly the right number of `int`
fields matching the copy width. For counter-reload: write the loop bound as a
direct memory read in the `for` condition (`i < *(int*)(base+off)`), not a cached
local, or the compiler will hoist it to a register and the loop will not reload.
